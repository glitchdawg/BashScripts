## REQUIREMENTS

```
sudo apt-get install ghostscript
sudo apt-get install figlet
sudo chmod 774 pdfcompz.sh
```

install this package or run requirements.sh

## USAGE

put your pdf file inside this directory with .pdf extension
and run pdfcompz.sh

```
./pdfcompz.sh
```
